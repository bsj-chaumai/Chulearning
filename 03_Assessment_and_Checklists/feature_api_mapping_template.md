# 機能×API・工数対応表（feature_api_mapping）

> **任意テンプレ。** ハイブリッド／API 連携型の案件で、**機能 ↔ 利用 API ↔ 実装方式 ↔ Phase** を 1 枚で見える化する。
> Step 03 の「漏れ逆引き」と Step 06 の「Phase 振り分け」に効く。実案件の検証で有効性を確認（v0.0.2 で追加）。

## 使い方
- 機能領域ごとに、利用する API（タグ）と本数、各 Phase の工数（人日）、実装方式を 1 行で表す。
- **実装方式**: ネイティブ / WebView / 外部ブラウザ / （ハイブリッド境界）。
- **★スコープ未確定**: API はあるが要件未確定の領域は `★` を付け、工数 0 で行を残す（一覧から消さない＝後で別途見積もり）。

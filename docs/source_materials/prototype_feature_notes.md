## 確定事項（実装済み）

### 画面構成（3タブ）
1. **卸価格管理（QUẢN LÝ GIÁ SỈ）**
   - 入力: 商品名 / 商品画像(file→Base64) / 仕入JPY / 為替 / 日本→ベトナム送料 / 梱包費
   - 計算: 原価 = JPY×為替 + 送料 + 梱包
   - 卸区分3段階（係数 1.17 / 1.136 / 1.102、÷0.98、1000 VND 丸め）
     - 小口 10–20 / 中口 30–40 / 大口 ≥50
   - 利益表示・一覧保存（画像サムネ付き）・削除
2. **小売商品管理（QUẢN LÝ SẢN PHẨM LẺ）**
   - 入力: 商品名 / 原価VND / Shopee売価
   - 利益概算: `売価 - 売価×25.4% - 原価`（手数料一式を 25.4% で近似）
   - 一覧保存・削除
3. **詳細料金・利益分析（TÍNH GIÁ & PHÍ CHI TIẾT）**
   - 入力: 仕入原価 / Shopee 売価
   - 内訳: 固定手数料14% / インフラ3,000đ / Voucher Xtra 4% / 決済約4.9% / GTGT 1% / TNCN 0.5% / 原価
   - 実受取利益の表示

### 技術特性
- 純クライアント（HTML + CSS + JS）
- 永続化: `localStorage`（`DB_SI` / `DB_LE`）
- 外部 API・認証・マルチユーザー・在庫・注文連携なし
- レスポンシブは簡易（max-width 1000px）
- ブランド色は Shopee オレンジ系

# Step 05: 金額構成 & 採算

## 1. 目的

Step 04 で積んだ明細を、**集計（フェーズ小計 → 管理費 → バッファ → 値引き → 税 → 合計）**にまとめ、**採算（社内: 原価・利益率）**を確認する。
「盛り（バッファ）」と「儲け（利益率）」を**見える化**し、提出に出す数字と社内で見る数字を分ける（Rule 4 / Rule 5 / Rule 10）。

## 2. 前提条件

- Step 04 完了（明細・見積もり設定が確定）。
- 参照: `00_AI_Instructions/pricing_and_rates.md`（採算の考え方）、`02_Estimation_Template/`（社内用 / 提出用レイアウト）。

## 3. ヒアリング項目

- 採算の基準: 目標利益率や下限ライン（社内基準。提出物には出さない）
- 値引きの方針: 値引き枠があるか（第一候補は値引きでなく**スコープ調整**）
- 複数案の要否: Phase 別 / スコープ別（Must だけ / Want 込み）/ バッファ別 を併記するか

## 4. 作業内容

1. **集計を組む（見える化）**:
   - フェーズ別小計 → **管理費（率）** → **バッファ（係数 or 上乗せ）** → **体制維持費（任意）** → **値引き（任意・明示行）** → 小計(税抜) → **消費税** → 合計(税込)。
   - 各ステップを行として明示する（隠さない・後付けしない＝ Rule 5）。
   - 契約を分けて請ける案件（上流=準委任 / 下流=請負 / その他）は、**契約区分ごとの小計**も併記する（区分で契約・検収が分かれるため。estimate_master ⑤ の「契約区分別 小計」）。
2. **採算を出す（社内用のみ）**:
   - 社内原価（オフショア原価 + 本社原価 + 管理）を積み、**利益率 = (社外金額 − 社内原価) ÷ 社外金額** を算出。
   - **バッファあり / なしの利益率を併記**して、値引き余地と下限を見せる。
   - 原価・利益率・社内単価は**社内用にだけ**載せる（Rule 10）。
3. **複数案（必要なら）**: 「Must だけ」「Want 込み」「Phase1 / フル」などを併記し、金額差を見せる（スコープ調整の土台）。
4. **スコープ調整シミュレーション**: 予算超過なら、機能 ON/OFF・Phase 分割で金額がどう動くかを示す（値引き前の第一候補）。
5. `🔔` 人間レビューゾーン（**金額・採算の確定**）で承認を得る。

> 値引き要求への構え: 「○○円下げる」ではなく「この機能を Phase2 に回すと○○円下がる」と**スコープで会話**できる状態を作る（Step 06 と連動）。
